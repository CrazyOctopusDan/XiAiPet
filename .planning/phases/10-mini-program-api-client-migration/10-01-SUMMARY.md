## Accomplishments

- Added `wx.request`-based customer API client with `xiaipet.customer.apiSession`, `Authorization: Bearer <token>`, login via `/api/v1/customer/auth/login`, normalized API errors, and one retry after `401`.

- Preserved catalog/runtime-config local fallback behavior so existing pages can keep rendering while later plans migrate checkout/order flows.
