<objective>
Add the customer miniapp HTTP API client, then migrate customer bootstrap, phone binding, catalog and runtime-config services away from CloudBase calls.
</objective>

<threat_model>
T10-01A: A customer token is missing, stale, or leaked into arbitrary request bodies.
Severity: high.
Mitigation: Store only `{ token, expiresAt, openid }` in `xiaipet.customer.apiSession`; inject token only through the `Authorization` header; clear storage after failed re-login.

T10-01B: Customer services silently continue to call unverified CloudBase functions after HTTP migration.
Severity: medium.
Mitigation: Replace defaults in migrated services with the shared HTTP client and add tests/grep checks for removed `wx.cloud.callFunction` usage.
</threat_model>
