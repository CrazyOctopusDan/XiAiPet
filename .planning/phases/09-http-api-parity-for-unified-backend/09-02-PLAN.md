<objective>
Expose customer-facing read APIs for catalog and runtime configuration under `/api/v1/customer`.
</objective>

<threat_model>
T9-02A: Customer catalog APIs leak merchant-only draft/admin product fields.
Severity: medium.
Mitigation: Route handlers call customer-specific service methods that filter to active/customer-safe fields.

T9-02B: Runtime config read exposes private payment or ops settings.
Severity: high.
Mitigation: Customer runtime config service allowlists public sections and strips private values.
</threat_model>
