<threat_model>
  <assets>
    - CloudBase 环境标识
    - users / merchant_users / runtime_configs 集合
    - 数据库安全规则和索引
  </assets>
  <entry_points>
    - cloudbaserc
    - env example files
    - collection/index/security config
  </entry_points>
  <threat severity="high">prod 环境被默认使用或与 dev 混淆，导致正式数据污染。</threat>
  <threat severity="high">前端可以直接写用户/白名单集合，绕过审计与鉴权。</threat>
  <mitigations>
    - `cloudbaserc.json` 只声明 dev/prod。
    - `database.rules.json` 强制敏感写经云函数处理。
    - `merchant_users` 独立集合承载白名单，不混入 runtime configs。
  </mitigations>
</threat_model>
