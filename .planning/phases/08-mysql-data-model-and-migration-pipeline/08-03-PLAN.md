<objective>
Create repository modules and transaction-safe service operations that future HTTP routes can call without coordinating raw multi-table writes.
</objective>

<threat_model>
T8-03: Route handlers later bypass transaction boundaries and create inconsistent order, stock, payment or balance state.
Severity: high.
Mitigation: Phase 8 exposes explicit transaction service functions and tests their atomic behavior before Phase 9 routes are implemented.
</threat_model>
