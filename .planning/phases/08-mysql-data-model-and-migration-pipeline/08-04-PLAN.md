<objective>
Build optional/idempotent CloudBase JSON import tooling and a verification report for seeded or imported MySQL data.
</objective>

<threat_model>
T8-04: Import tooling duplicates balances/orders or trusts malformed exported data.
Severity: high.
Mitigation: importer validates input shape, upserts by legacy IDs/openid/idempotency keys, and verification reports duplicate or inconsistent financial state.
</threat_model>
